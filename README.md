В проекте игра с логикой головоломки
Сделана для тестового задания и обучения простых механик взаимодейский между авторами
НА WSAD перемещать персонажа
на пробел прыжок
есть механики открывания дверей при взаимодейсвии с площадками
возможно взаимодействие с предметами
1 - нажимать Е и включать 
2 - жать Е и открывать меняя поворот предметов
3 - подходить и забирать предметы
Суть игры найти неочевидный выход что и есть сама головоломка))))
